# Mini Projet Python – Gestion et analyse des notes des étudiants

## Description
Ce mini-projet Python permet de gerer et analyser les notes d’etudiants a partir
d’un fichier CSV à l’aide d’une interface console simple.

---

## Fonctionnalites
- Interface console avec menu interactif
- Calcul de la moyenne générale
- Calcul de la médiane
- Calcul de l’écart-type
- Recherche de la meilleure note
- Recherche de la pire note
- Liste des étudiants admis (note > 10)
- Liste des étudiants ajournés (note < 10)

---

## Notions utilisees
- Programmation orientée objet (POO)
- Lecture de fichiers CSV
- Statistiques descriptives
- Structures conditionnelles et boucles
- Interaction utilisateur (console)

---

## Technologies
- Python 3
- Bibliothèques standards : csv, math

---

python main.py
